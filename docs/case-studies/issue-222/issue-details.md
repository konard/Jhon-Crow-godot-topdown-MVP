## Original Description (Russian)
три шага:
1. достать магазин левой рукой с груди
2. вставить в винтовку
3. передёрнуть затвор

## Translation (English)
Three steps:
1. Take the magazine with the left hand from the chest
2. Insert into the rifle
3. Pull the bolt/charging handle

## Key Requirements
1. **Magazine Retrieval Animation**: Left arm moves to chest area to grab a new magazine
2. **Magazine Insertion Animation**: Left arm brings magazine to the weapon and inserts it
3. **Bolt Pull Animation**: Character pulls the charging handle to chamber a round

## Technical Context
This is a top-down 2D game using Godot 4 with a hybrid C#/GDScript approach.
